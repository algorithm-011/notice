## 앞으로의 계획

알고리즘과 자료구조는 매주 1개 이상의 개념을 주제로 어떤 알고리즘/자료구조인지, 어떻게 동작하는지, 실제 구현한 코드에 대해서 조사한 후 github에 올린다.

문제풀이는 매주 영어로 된 문제와 한국어로 된 문제를 각각 1개 이상 풀고, 문제풀이에 대한 간략한 설명과 함께 코드를 github에 올린다. 사용 언어는 Python3, C, C++ 셋 중 하나로 제한한다.
